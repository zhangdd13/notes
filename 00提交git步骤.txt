1、先在本地同步一下仓库的内容,命令如下：
git pull git@github.com:zhangdd13/notes.git

2、下面就要把创建的文件上传到到仓库上去了，首先执行增加命令，如下：
git add .    （这后面是一个英文的句号）
add后面加了一个点，是想要提交所有文件。如果想提交指定的文件，可以写文件名。

3、执行完增加命令后，要执行提交命令，如下：
git commit Cm “这里写下你自己的记录本次提交内容的信息”

-m后面跟提示信息，这个提示信息是一定要写的，不仅是规则，同时也方便我们记录我们提交的过程，写清晰为什么提交或修改了什么是非常有用的

4、提交完成后，我们就要把它推送到远程仓库上去了，命令如下：
git push git@github.com:zhangdd13/notes.git

5、这样就完成了我们要做的所有任务
